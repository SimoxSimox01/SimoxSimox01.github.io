Ik ben Simon uit 4H4

Ik kom uit Nieuw-Vennep en ga hier ook naar school, naar het HVC. Ik heb daar vijf dagen in de week les. Op dindagochtend heb ik van 8:20 tot 10:00 het vak informatica, ook heb ik nog een uur informatica op vrijdagmiddag van 14:10 tot 15:00. Voor dat vak maak ik nu ook deze website!

In mijn vrije tijd maak ik graag muziek, dan bespeel ik de drum of mijn saxofoon. Ik drum nu 8 jaar en saxofoon bespeel ik nu een half jaar. Verder zwem ik ook nog voor mijn lol. Ook game ik graag op mijn computer. Thuis maak ik ook vaak games voor het online platform Roblox, dat doe ik met de code LUA. In het weekend werk ik op zaterdag van 7 tot 3, verder zit ik ook vaak te leren of huiswerk te maken.
